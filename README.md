# TextUtils - A Text Manipulating App

TextUtils is a simple React JS app that offers these features:

- Transform text to upper or lower case.
- Trim sentence whitespace.
- Remove special characters.
- Clear text easily.
- Count words and letters.
- Estimate reading time for content.

## Getting Started with Text Utils

- Clone/Download code from GitHub to your local repository.
- Open the folder in VS Code or any other IDE.
- Open Terminal and write `npm start` to run app in the development mode.

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
You may also see any lint errors in the console.
